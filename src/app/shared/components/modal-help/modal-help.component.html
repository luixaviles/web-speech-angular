Available Actions

Continuous Dictation

Just press the microphone and start talking. You should be able to see the text result into the main textarea. Try to write a letter with your voice!

Change the Application Theme

Once the microphone is active, try the following voice commands:

English
Voice Command Action
"Perform Change Theme" Enables the Change Theme Strategy
"Finish Change Theme" Finishes the Change Theme Strategy

Available theme names: "deep purple", "indigo", "pink" and "purple"

Español
Comando de Voz Acción
"Iniciar Cambio de Tema" Habilita la Estrategia para el cambio de Tema
"Finalizar Cambio de Tema" Finaliza la Estrategia para el cambio de Tema

Nombres de tema disponibles: "púrpura", "azul", "rosa" and "verde"
